# FactoryPulse – Real-time Industry 4.0 Monitoring

Rust + Next.js ile gerçek zamanlı fabrika izleme dashboard'u. 50 sanal IoT cihazı veri gönderir, WebSocket ile canlı grafik güncellenir.

## Özellikler
- Real-time data stream (WebSocket + RabbitMQ)
- Live charts (Recharts)
- 7-container Docker setup
- PostgreSQL + Redis backend
